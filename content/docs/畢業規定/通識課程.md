- 通識課程開課代碼為 GE，核心通識課程開課代碼為 GEC
- 一學期選課至多選到3門通識，但志願序可填20個。若需要修習超過3門，可用加簽方式。
- 修習規定
    - 畢業前需修習滿20學分通識課程
    - 核心通識須修8-12學分，並四個向度各至少一門，選修通識也是8-12學分
    - 核心通識的四個項目有：
    1. 思維、文明與歷史
    2. 生命、自然與科技
    3. 藝術、文學與創意設計
    4. 公民、社會與世界

2. 通識課是只要選自己喜歡的就好？
    - 要注意旁邊是否有選課對象限制。
    - 雖然多半都是開給全校學生選修的，但有些課會註明開課學院/系所的學生修習後沒有辦法列入通識學分，所以要細看一下。
